## Full-Stack Workshop 1 – Git Merge Conflict Activity
### (https://docs.google.com/document/d/1ai74BqukeBx3Z8MgOsK3nCUF2ag9DgBGky2aC7tW8R8/edit?usp=sharing)
This repository is for an Intro to Web Dev Workshop1 (3/12/26) activity on Git collaboration. The students intentionally create a merge conflict in `menu.txt` by changing the same line in different copies of the repo. Students then resolve the conflict using the standard Git workflow: `git status`, `git pull`, manually editing conflict markers, `git add`, `git commit`, and `git push`. Along the way, they also practice basic command-line navigation using `ls` to list folders and `cd` / `cd ..` to move into the project directory and back out again.
